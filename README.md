## f1-service-backend

### Setup the project

<pre>
#1 copy .env.example 

cp .env.example .env
</pre>

<pre>
#2 install dependecies 

npm install
</pre>

<pre>
#3 build the project

npm run build
</pre>

<pre>
#4 start the project

npm start
</pre>

### Or use it with docker
<pre>
docker-compose up -d
</pre>
